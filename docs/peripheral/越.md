# 越- NPs (classical)

??? note "Bibliography"

    楊玉鋒, '《岳陽樓記》「越明年」訓釋定讞與岳陽樓的增修時間', 寧波大學學報, 35/4 (2022), 100–5.

When the NP is a shifter (e.g. 明年) or definite time specification (e.g.
四月十二日), the proclitic 越- marks it as the temporal location of the clause.
When the NP represents a duration (e.g. 三日), on the other hand, 越- marks it
as the length of the period between the time of the clause and that of the
previous one.
